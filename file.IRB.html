Interactive Ruby Shell

Starting IRB

You can develop machine vision algorithms using the Interactive Ruby Shell. Simply start the shell like this:

irb

You can put frequently used command into the file $HOME/.irbrc. It is useful to load Ruby Gems, IRB command line completion, and syntax highlighting (see FAQ for more information on wirble).

require 'irb/completion'
require 'wirble'
Wirble.init( { :skip_prompt => true } )
Wirble.colorize
